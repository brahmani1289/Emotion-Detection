# Emotion Detection Project
This project predicts emotions (Angry, Fear, Happy, Normal, Sad) using Heart Rate and Age data.

## Steps to Run
1. Install requirements: `pip install pandas scikit-learn joblib seaborn matplotlib`
2. Run `Emotion Detection.py`
3. Example prediction: `(80, 25) → Normal`
